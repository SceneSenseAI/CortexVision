# Requirements Layout

core.txt
General utilities.

vision.txt
Shared computer vision dependencies.

Grounded DINO, YOLO, object detection.

depth.txt
Depth Anything and depth estimation.

flow.txt
RAFT and optical flow.

segmentation.txt
SAM and segmentation.

tracking.txt
BoT-SORT and tracking.

pose.txt
DWPose and pose estimation.

scene_graph.txt
Scene graph generation.

audio.txt
Whisper and audio event detection.

temporal.txt
VideoMAE, SlowFast, temporal reasoning.

vlm.txt
GPT, Claude, LLaVA integrations.

dashboard.txt
Gradio and FastAPI.

evaluation.txt
Metrics and reports.

dev.txt
Testing, linting, notebooks.

full.txt
Installs everything.
